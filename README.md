
# Dermatology App UI

This Flutter project is a UI design for a dermatology app dashboard. It includes the following UI components:

## UI Components:

### 1. Next Appointment
- A visually appealing section that displays the patient's next appointment date and time.
- Includes a reminder/notification feature for the patient.

### 2. Doctor's Details
- A comprehensive profile of the dermatologist, showcasing their qualifications, experience, and specialization.

### 3. Patient Ratings and Reviews
- A system for patients to rate their experiences and leave reviews.
- Utilizes visual elements such as star ratings and patient testimonials.

### 4. Current Patient Information
- A section displaying the name and basic information of the patient currently being seen by the dermatologist.
